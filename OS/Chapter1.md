# Operating System definitions
Operating System은 Software이다.    
1. Computer Hardware를 관리해주는 프로그램이다.
2. 다른 여러 프로그램들을 실행할 수 있도록 조정해주는 프로그램이다.

## The General Roles of the OS
- Hardware management 
    - Access to I/O devices
    - Access to files
    - Accounting
    - Error detection
- Program execution
    - Scheduling
    - Error reportin
